什么是 MedMamba？

MedMamba 是一种专为医学图像分析设计的深度学习架构，其核心基于 Mamba — 一种新兴的状态空间模型（State Space Model, SSM）框架。 它在保持强大图像理解能力的同时，相比 Transformer 架构具有更低的计算复杂度，尤其适合处理高分辨率医学影像。

⚡
线性复杂度
O(n) 序列处理复杂度，比 Transformer 的 O(n²) 高效
🌐
全局感受野
每个位置都能关注全局上下文，不受局部窗口限制
🏥
医学专项
针对血细胞、病理切片等医疗影像进行优化

状态空间模型（SSM）

状态空间模型是一类将连续序列映射到输出的数学框架，其核心方程如下： 

h'(t) = A · h(t) + B · x(t)    # 状态转移方程
y(t)  = C · h(t) + D · x(t)    # 输出方程

# 其中：
# x(t): 输入信号（图像 patch 序列）
# h(t): 隐状态（记忆向量）
# y(t): 输出特征
# A, B, C, D: 可学习参数矩阵

Mamba 在此基础上引入了选择性机制（Selective SSM）， 使模型能够根据输入内容动态调整状态转移矩阵，解决了传统 SSM 无法进行内容感知的问题。
